addHandlerToRange(event, func, start, end)

Adds an event listener to run the given function in the days within the given range
Parameters:
Name Type Description
event string the name of the event as a string, e.g. "mouseenter"
func function the function to handle the event
start number the number of the first day in range
end number the number of the last day in the range
Source:
  • scriptCalendar.js, line 167
Example
- All days in the calendar have an id in the format "day-<number>".
This function uses the number part as number, not string:
addHandlerToRange("mouseenter", hoverInAfterCheckin, 34, 38); 
Will  add a listener to run the function when the mouse enters 
the element for the days 35, 36 and 37.
